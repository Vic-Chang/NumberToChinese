# 中文大寫數字 與 阿拉伯數字轉換 NumberToChinese

## 功能

用於方便某些場合必須使用大寫正體中文書寫時使用

## 使用限制

阿拉伯數字轉正體中文大寫最大數字 : 九千兆以下(一京以下)

正體中文大寫最大數轉字阿拉伯數字 : 九千億以下(一兆以下)
